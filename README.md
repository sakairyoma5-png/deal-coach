# DealCoach

AI sales training SaaS for roleplay practice, coaching feedback, skill tracking, and team-based sales enablement.

## Overview

DealCoach is a web application designed to help salespeople improve through AI-powered roleplay simulations and structured feedback.

The app supports individual training and team-based management, allowing users to practice sales conversations, receive AI feedback, track skill progress, and manage subscription plans.

## Features

- AI sales roleplay simulations
- Automated coaching feedback
- Skill card generation
- User authentication
- Team and organization management
- Admin dashboard
- Stripe subscription payments
- Premium plan and billing flow
- Progress tracking and training logs

## Tech Stack

- TypeScript
- React
- Vite
- Express
- PostgreSQL
- Drizzle ORM
- OpenAI API
- Stripe
- Replit Auth

## Why I Built This

I built DealCoach based on my experience in solution sales.

Sales training often depends on senior staff, managers, or limited roleplay opportunities. DealCoach was created to make sales practice more repeatable, measurable, and accessible through AI.
